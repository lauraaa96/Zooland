API RESTful

ANIMALES

GET /api/animales Obtener todos los artículos.
POST /api/animales Añadir un animal.
GET /api/animales/{id} Obtener un animal.
UPDATE /api/animales/{id} Modificar un animal.
DELETE /api/animales/{id} Eliminar un animal.
